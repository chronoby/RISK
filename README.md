# RISK

An 3D scene demo in OpenGL.

### Features

- Transformations
- Shaders
- Textures
- Lights
- Model loading
- Shadow mapping
- Skybox

### Dependencies

- GLFW 3.3.3
- GLM
- GLAD
- Dear ImGui
- assimp 5.0.1
- stb
